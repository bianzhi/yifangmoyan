# 缠论三类买卖点：原文定义与实现对照

## 总览

缠论买卖点分为三类，对应第17课、20课、21课、101课原文定义。三类买卖点都被理论保证为"100%安全的买卖点"。

| 类型 | 产生条件（买点） | 核心原文 | 实现函数 |
|------|------------------|----------|----------|
| 第一类买点 | 下跌趋势（≥2个中枢）底背驰，在最后一个中枢之下 | 17课、21课 | `find_buy1_sell1` |
| 第二类买点 | 一买后，第一次回抽的低点 | 21课、101课 | `find_buy2_sell2` |
| 第三类买点 | 中枢之上，次级别离开 + 次级别回抽不跌破ZG | 17课、20课、21课 | `find_buy3_sell3` |
| 二三买重合 | 一买后凌厉上破最后中枢，回抽不触及中枢 → 2、3买重合 | 21课 | `merge_overlapping_2_3` |

---

## 一、第一类买卖点

### 原文（21课）

> "只有在下跌确立后的中枢下方才可能出现买点。这就是第一类买点。"

### 原文（17课）

> "趋势背驰产生第一类买卖点"

### 原文（27课）

> "盘整背驰……转化为第三类买卖点"（不是第一类！）

### 核心要素

| 要素 | 原文依据 | 实现 |
|------|----------|------|
| **趋势条件** | ≥2个同级别中枢，同方向递进 | `group_zs_indices_by_trend` 分组，每组≥2个中枢才构成趋势 |
| **背驰条件** | 趋势背驰（趋势底背驰→一买，趋势顶背驰→一卖） | 只取 `bc_sub_type == "trend"` 的背驰，盘整背驰不产生一类买卖点 |
| **位置条件** | 在中枢之下（下跌趋势最后一个中枢下方） | 背驰点之前的最近中枢：下跌趋势中 `bd.direction == "down"` + `trend_dir == "down"` |
| **铁律** | 盘整背驰绝对不能产生第一类买卖点 | 明确过滤 `bc_sub_type != "trend"` |

### 实现逻辑（`find_buy1_sell1`）

```
1. 遍历所有趋势背驰点
2. 找到背驰点之前结束的所有中枢
3. 将中枢按方向递进分组（同向不重叠为同一趋势）
4. 对每组≥2个中枢的趋势：
   - 下跌趋势 + 底背驰 → 一买
   - 上涨趋势 + 顶背驰 → 一卖
5. 一买/一卖价格 = 背驰点的段终点价格
```

## 二、第二类买卖点

### 原文（21课）

> "第二类买点是和第一类买点紧密相连的，因为出现第一类买点后，必然只会出现盘整与上涨的走势类型，而第一买点出现后的**第二段次级别走势低点**就构成第二类买点"

> "第二类买点，不必然出现在中枢的上或下，可以在任何位置出现"

### 原文（101课·答疑1）—— 二买的三种情况

> **一、最强的情况**
> 第二类买点刚好构成原来下跌的最后一个中枢开始的震荡走势的第三类买点，也就是第二、三类买点合一了，这是最强的走势，这情况，一般都对应V型反转的快速回升，是最有力度的。

> **二、最弱的情况**
> 第二类买点跌破第一类买点，也就是第二类买点比第一类买点低，这是完全可以的，这里一般都构成盘整背驰

> **三、一般性走势**
> 也就是前面两者之间的走势。这种情况下，第一、二、三买点，是依次向上，一个比一个高。

### 核心要素

| 要素 | 原文依据 | 实现 |
|------|----------|------|
| **前置条件** | 必须先有一买/一卖 | 遍历 `first_points` 中 `bs_type == "1buy"` 的点 |
| **构造** | 一买后的第一段向上（离开段）+ 第二段向下（回抽段），回抽段终点=二买 | `find_buy2_after_buy1`：先找第一个up段，再找紧随的down段 |
| **不破一买价** | 101课明确允许跌破一买（最弱情况） | 区分两类：`pullback_low >= buy1.price` → `"2buy"`，`<` → `"2buy_break"` |
| **位置** | 原文"可以在任何位置出现" | 只找紧挨一买后的第一个up→down对，不限定位置 |

### 实现逻辑（`find_buy2_after_buy1`）

```
1. 从一买位置开始，找到第一个向上的段（离开段）
2. 然后找紧随的向下段（回抽段）
3. 回抽段的终点低点 = 二买位置
4. 区分标准二买(不破一买价) vs 破位二买(破了一买价)
```

### 101课二买三种情况的当前实现状态

| 情况 | 原文 | 已实现 |
|------|------|--------|
| 最强（二三买重合） | 回抽低点在中枢上方 | ✅ `merge_overlapping_2_3` |
| 最弱（破一买） | 回抽低点 < 一买价 | ✅ `2buy_break` |
| 一般性 | 一二三买依次向上 | ✅ `2buy`（标准二买） |

## 三、第三类买卖点

### 原文（20课·定理）

> **缠中说禅第三类买卖点定理：** 一个次级别走势类型向上离开缠中说禅走势中枢，然后以一个次级别走势类型回试，其低点不跌破ZG，则构成第三类买点；一个次级别走势类型向下离开缠中说禅走势中枢，然后以一个次级别走势类型回抽，其高点不升破ZD，则构成第三类卖点。

### 原文（21课）

> "第三类买点是中枢扩张或新生产生的。中枢扩张导致一个更大级别的中枢，而中枢新生，就形成一个上涨的趋势，这就是第三类买点后必然出现的两种情况。"

> "一个上涨趋势确定后，不可能再有第一类与第二类买点，只可能有第三类买点。"

### 核心要素

| 要素 | 原文依据 | 实现 |
|------|----------|------|
| **离开段** | 必须实际脱离中枢 | 三买：离开段高点 > Zg；三卖：离开段低点 < Zd |
| **回抽段** | 次级别走势回试，不破ZG/不升破ZD | 三买：回抽低点 >= Zg；三卖：回抽高点 <= Zd |
| **"="处理** | 原文"不跌破ZG"含等于 | 边界用 `>=` 和 `<=`（有专门测试验证） |
| **段对** | 离开和回抽必须是完整走势段 | 扫描中枢后所有相邻段对 `(after_segs[i], after_segs[i+1])` |
| **定位** | 回抽段终点 = 买卖点位置 | `index = back_seg.end_idx`, `price = back_low/high` |

### 实现逻辑（`find_buy3_sell3`）

```
对每个中枢：
  取中枢结束后的所有段
  扫描所有相邻段对 (离开段, 回抽段)：
    三买条件：
      离开段方向=up AND 离开段高点 > Zg  （脱离中枢上沿）
      AND 回抽段方向=down AND 回抽段低点 >= Zg  （回试不破Zg）
    三卖条件：
      离开段方向=down AND 离开段低点 < Zd  （脱离中枢下沿）
      AND 回抽段方向=up AND 回抽段高点 <= Zd  （回抽不升破Zd）
```

## 四、二三买重合

### 原文（21课）

> "只有第二类买点与第三类买点是可能产生重合的，这种情况就是：第一类买点出现后，一个次级别的走势凌厉地直接上破前面下跌的最后一个中枢，然后在其上产生一个次级别的回抽不触及该中枢，这时候，就会出现第二类买点与第三类买点重合的情况"

> "一旦出现这种情况，一个大级别的上涨往往就会出现。"

### 实现逻辑（`merge_overlapping_2_3`）

```
在买卖点列表中，同一index同时存在：
  - 2buy/2buy_break + 3buy → 合并为 "2+3buy" / "2+3buy_break"
  - 2sell/2sell_break + 3sell → 合并为 "2+3sell" / "2+3sell_break"
```

## 六、已知局限与待改进项

### 1. 二买检测的段级别问题
- **当前实现**：二买在同级别段上找"up→down"对，段本身未进一步分解为次级别
- **原文语义**：21课和101课明确说二买是"第一类买点的**次级别**回抽"产生的
- **影响**：如果分析级别是30分钟笔，则二买应该在5分钟笔上找。当前同级别查找会过度检测。
- **方向**：需要实现跨级别的二买检测（用递归后的次级别走势）

### 2. 三买的离开段"次级别"验证
- **当前实现**：只要是离开+回抽段对即可，未验证离开段内部是否完成了次级别走势类型
- **原文语义**：20课强调"次级别走势类型"离开和回试
- **影响**：可能会把简单的线段波动误判为三买

### 3. 一买的位置约束（中枢之下）
- **当前实现**：只验证了趋势方向和背驰方向匹配，未显式验证一买是否在最后一个中枢下方
- **原文语义**：21课"下跌确立后的中枢**下方**才可能出现买点"
- **影响**：理论上一买可能出现在中枢区间内，但趋势底背驰点天然在下方，差异不大
